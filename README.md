🌤️SkyCast Weather 

A simple and modern weather web app built with (HTML, CSS, and JavaScript).  
It shows the current, hourly, and daily weather forecast with real-time updates.

✨ Features
  - Detect your location automatically
  - Search by city, state, or country (English preferred)  
  - Current, hourly (24h), and 7-day forecast
  - Dynamic background and theme based on weather
  - °C / °F toggle
  - Responsive design  

Live Demo
👉 [View SkyCast Weather](https://ebaa-anas.github.io/SkyCast-Weather/)

🛠️ Technologies Used
- HTML5 
- CSS3  
- JavaScript   
- OpenWeatherMap API
